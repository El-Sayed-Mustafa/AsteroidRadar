#Project Overview
Asteroid Radar is an app to view the asteroids detected by NASA that pass near Earth

#Using 
 -NEoWs APi
 -RESTful 
 -Moshi 
 -Picasso 
 -Room database
 -Work Manager and constrain 
 -MVVM
 -Navigation
 -Recyclerview
 -Filtering
